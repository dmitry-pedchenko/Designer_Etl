Exception - Исключение при котором программа не останавливается
Error     - Ошибка при которой программа не может продолжать работу и останавливается

-----------------------------------------------------------------------------------------------------

Exception 1 - Исключение возникшее при отправке запроса на базу, какая-то ошибка при выполении запроса. Выводится
ответ от базы данных
текст исключения: Exception while commiting query: row number:{номер строки}
на следующей строке: Query: {запрос}
на следующей строке: {ответ от базы}


Error 1 - Ошибка чтения конфига
текст ошибки: Error at parsing XML configuration file

Error 2 - Не может найти конфиг
текст ошибки: Cant find file XML {путь по которому ищет}
на новой строке: причина по которой не открылся файл

Error 3 - Ошибка парсинга конфига, нет необходимого тега в блоке описания данных подключения к базе данных
текст ошибки: Can't find tag {имя тега которого нет} in xml

Error 4 - Ошибка парсинга конфига, нет необходимого тега в блоке описания колонок в экселе.
текст ошибки: Can't find tag {имя тега которого нет} in {в каком блоке не может нафти} tag at block number {номер блока по порядку в блоке importXml в конфиге} in importXml/columns block in xml


Error 5 - Ошибка парсинга конфига, нет необходимого атрибута mode="" у тега в блоке описания колонок в экселе.
текст ошибки: Can't find property mode in tag {имя тега у которого нет атрибута} in {блок в котором находится тег} tag at block number {номер блока по порядку в блоке importXml в конфиге} in importXml/columns block in xml


Error 6 - Ошибка парсинга конфига, нет необходимого тега в блоке описания полей в базе данных.
текст ошибки: Can't find tag {какой тег не может найти} in {в каком блоке} tag at block number {номер блока по порядку в блоке exportTable в конфиге} in exportTable block in xml


Error 7 - Ошибка парсинга конфига, нет необходимого атрибута mode="" у тега в блоке описания полей в базе.
текст ошибки: Can't find property mode in tag {тег у которого нет атрибута} in {блок в котором тег} tag at block number {номер блока по порядку в блоке exportTable в конфиге} in exportTable block in xml


Error 8 - Не может открыть эксель
текст ошибки: Can't open Excel path: {путь по которому ищет} on page: {номер страницы, номер страниц начинается с 1}


Error 9 - Ошибка подключения к базе данных
текст ошибки: Fail to connect to {хост} {имя базы} {пользователь} {имя таблицы} {порт}



Error 10 - Ошибка выполнения запроса к базе данных в процессе выполнения проверки корректности данных
текст ошибки: Error in exec query validate {на следующей строчке ответ базы данных о причинах ошибки}


Error 11 - Поле экселя описанное в конфиге в блоке importXml не существует в экселе
текст ошибки: Column: {имя поля} not exists in {имя экселя}. You must chose from this names: {список полей которые есть в экселе}


Error 12 - Не хватает описания полей в блоке описания полей в бозе данных exportTable. В базе поле есть а в конфиге нет
текст ошибки: Columns: {поле которого нет} not exists in exportTable/columns list in config



Error 13 - Поле базы данных описанное в блоке exportTable не существует в таблице
текст ошибки: Columns: {поле описанное в конфиге} not exists in table {имя таблицы в которой нет поля}; In DB: {имя базы данных}



Error 14 - В конфиге для поля в базе данных в блоке exportTable нет источника в блоке importXml.
Если у поля базы данных в описании стоит свойство <fromExcel>true</fromExcel> то система ищет его в
блоке описания полей источника по тегу <colNameDb>имя поля в базе</colNameDb>
Если тег <fromExcel>false</fromExcel> то система не будет искать это поле в полях источника.
Если у поля тег <isAutoInc>false</isAutoInc> то поле так же ищется в списке источников
текст ошибки: Columns: {поле для которого нет источника} not exists in importXml/columns/column/colNameDb tag in the {имя конфига} file configuration.
на следующей строчке: List of exists columns in that tag: {список полей которые описаны}



Error 15 - Не выбран режим запуска
текст ошибки: Run mode don't selected



Error 16 - Не может округлить число, скорее всего попала строка. Возникает из-за того что
тег <colType>int</colType> в поле описания колонки ждет число а пришла строка
текст ошибки: Can't round number {значение которое не может округлить}
